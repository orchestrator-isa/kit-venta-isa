# ONBOARDING — Día 0 al Día 30
## Protocolo de entrega y seguimiento

---

## MINUTO 0 (Firma del contrato)
- [ ] Enviar mensaje de bienvenida (plantilla: bienvenida-cliente.txt)
- [ ] Solicitar materiales: fotos, menú, horarios
- [ ] Agendar fecha de primera entrega
- [ ] Activar recordatorio automático (24h)

## DÍA 1 — ENTREGA 1A: FUNDAMENTOS
- [ ] Optimizar Google Maps
- [ ] Configurar WhatsApp Business
- [ ] Crear redes sociales
- [ ] Generar QR Code
- [ ] Enviar checklist 1A firmado
- [ ] Mensaje al cliente: "Tu Maps ya está activo. Búscate en Google."

## DÍA 3 — SEGUIMIENTO
- [ ] Verificar que el cliente aparece en Maps
- [ ] Preguntar: "¿Ya recibiste algún mensaje nuevo?"
- [ ] Si NO → Ajustar SEO, añadir más fotos, pedir primera reseña
- [ ] Si SÍ → Celebrar, documentar

## DÍA 7 — PRIMER REPORTE
- [ ] Generar reporte semanal (plantilla)
- [ ] Enviar por WhatsApp + email
- [ ] Llamada de 5 minutos: "¿Qué tal la primera semana?"

## DÍA 15 — REVISIÓN DE MÉTRICAS
- [ ] Revisar dashboard
- [ ] ¿+30% consultas? → Preparar upsell
- [ ] ¿Sin cambios? → Ajustar estrategia, más SEO, más reseñas

## DÍA 30 — EVALUACIÓN OFERTA GANCHO
- [ ] Enviar evaluación (plantilla: evaluacion-30-dias.txt)
- [ ] Si cliente confirma resultados → Activar cobro 200 MAD/mes
- [ ] Si cliente NO confirma → Ofrecer extensión 15 días o despedida amistosa
- [ ] Si cliente quiere escalar → Presentar Pack siguiente

---
**Frecuencia de contacto:**
- Semana 1: Cada 2 días
- Semana 2-3: Cada 3 días  
- Semana 4+: Semanal (reporte + llamada)
