# CHECKLIST DIAGNÓSTICO RÁPIDO — 5 MINUTOS

Cuando un cliente dice "no funciona", verificar en este orden:

## 1. Google Maps (1 min)
- [ ] ¿Aparece al buscar "[tipo] Tetuán"?
- [ ] ¿Ficha completa (fotos, horarios, teléfono)?
- [ ] ¿Reseñas recientes (último mes)?
- [ ] ¿Posición en top 10?

## 2. WhatsApp Business (1 min)
- [ ] ¿Número correcto y activo?
- [ ] ¿Perfil completo (foto, descripción, horario)?
- [ ] ¿Catálogo visible?
- [ ] ¿Mensaje de ausencia configurado?

## 3. Bot / Chatbot (1 min)
- [ ] ¿Responde en <3 segundos?
- [ ] ¿Menú principal funciona?
- [ ] ¿Flujo de pedido completo?
- [ ] ¿Derivación a humano funciona?

## 4. Dashboard (1 min)
- [ ] ¿Métricas actualizadas (última hora)?
- [ ] ¿Cliente tiene acceso?
- [ ] ¿ROI visible?

## 5. Materiales físicos (1 min)
- [ ] ¿QR pegado en puerta/mostrador?
- [ ] ¿Tarjetas entregadas a clientes?
- [ ] ¿Poster "Pide por WhatsApp" visible?

## Resultado
- **5/5 verdes** → Problema es percepción del cliente. Formación.
- **3-4 verdes** → Problema menor. Ajuste rápido.
- **0-2 verdes** → Problema grave. Intervención completa.
